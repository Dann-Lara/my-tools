# YouTube Module

## Overview

YouTube Channel Hack module for generating niches, managing channels, and creating content ideas with AI assistance.

## Endpoints

### Niches

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/v1/youtube/niches` | JWT | List all available niches (generated by AI) |
| DELETE | `/v1/youtube/niches/:id` | JWT | Delete a niche by ID |

### Channels

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/v1/youtube/channels` | JWT | List all channels for the current user |
| GET | `/v1/youtube/channels/:id` | JWT | Get a specific channel by ID |
| POST | `/v1/youtube/channels` | JWT | Create a new YouTube channel |

### Ideas

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/v1/youtube/channels/:channelId/ideas` | JWT | List all ideas for a channel |
| POST | `/v1/youtube/channels/:channelId/ideas` | JWT | Generate new ideas for a channel |
| GET | `/v1/youtube/ideas/:id` | JWT | Get a specific idea by ID |
| PATCH | `/v1/youtube/ideas/:id/seo` | JWT | Update idea SEO data |
| PATCH | `/v1/youtube/ideas/:id/metrics` | JWT | Update idea metrics |
| PATCH | `/v1/youtube/ideas/:id/status` | JWT | Update idea status |
| POST | `/v1/youtube/ideas/:id/script` | JWT | Generate script for an idea |
| POST | `/v1/youtube/ideas/:id/prompts` | JWT | Generate AI prompts for an idea |
| POST | `/v1/youtube/channels/:channelId/ideas/regenerate` | JWT | Regenerate all ideas for a channel |

### Monetization

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/v1/youtube/channels/:channelId/monetization` | JWT | Get monetization setup for a channel |
| PATCH | `/v1/youtube/channels/:channelId/monetization/steps/:stepId` | JWT | Update a monetization step |

## Entities

- **ChannelEntity**: YouTube channel with status tracking
- **ContentIdeaEntity**: Content ideas with SEO, metrics, and status
- **AIVideoPromptEntity**: AI-generated prompts for video creation
- **MonetizationSetupEntity**: Monetization checklist for channels
- **NicheEntity**: Generated niches with opportunity scores

## Permissions

Module key: `youtube`

Users need `youtube` permission to access this module. Configured via `allowedModules` in user entity.

## Frontend Pages

- `/admin/youtube` - Redirect to channels or niches based on existing channels
- `/admin/youtube/channels` - List all channels
- `/admin/youtube/niches` - Browse and generate niches, create channels
- `/admin/youtube/[id]` - Channel detail with tabs (ideas, prompts, monetization)
